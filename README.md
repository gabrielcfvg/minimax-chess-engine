
# ASCII Chess
For Windows

## Releases
* v0.1 -- initial release 

## How to play
```
s -- start the pieces
m -- move mode
r -- return to initial mode 
    
P4 d5 -- move piece
Piece<space>notation<enter>

K -- king 
Q -- queen 
R/r -- rooks 
B/b -- bishops
N/n -- knights
P1-8 -- pawns 
```
## Actions
```
s -- start pieces
e -- exit
m -- move mode
b -- back last moviment
r -- return to initial mode
c -- clear annotations

: -- initial mode 
w: -- white turn
b: -- black turn
```
