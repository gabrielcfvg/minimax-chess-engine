
# Chess
For Windows

## Releases
* v0.1 -- initial release 

## How to play
```
m -- move mode
t -- return to init mode 
    
P4 d5 -- execute move
Piece<space>notation<enter>

K -- king 
Q -- queen 
R/r -- rooks 
B/b -- bishops
N/n -- knights
P1-8 -- pawns 
```
## Actions
```
e -- exit
m -- move mode
a -- back move
t -- return to init mode
c -- clear notes

: -- init mode 
w: -- white turn
b: -- black turn
```
