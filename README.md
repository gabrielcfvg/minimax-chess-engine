
# Chess
For Windows

## Releases
* v0.1 -- initial release 

## How to play
```
s -- start
m -- move mode
r -- return to init mode 
    
P4 d5 -- execute move
Piece<space>notation<enter>

K -- king 
Q -- queen 
R/r -- rooks 
B/b -- bishops
N/n -- knights
P1-8 -- pawns 
```
## Actions
```
s -- start
e -- exit
m -- move mode
b -- back move
r -- return to init mode
c -- clear notes

: -- init mode 
w: -- white turn
b: -- black turn
```
